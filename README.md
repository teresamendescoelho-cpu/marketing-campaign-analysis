# Marketing Campaign Analysis

## Project Overview

This project analyzes marketing campaign performance using Python and Pandas.

The objective is to calculate and compare conversion rates across different campaigns and identify which campaign performs best.

## Technologies Used

- Python
- Pandas

## Metrics Analyzed

- Clicks
- Conversions
- Conversion Rate

## Example Output

| Campaign | Clicks | Conversions | Conversion Rate |
|----------|---------|------------|----------------|
| A | 1200 | 60 | 5.0% |
| B | 800 | 25 | 3.1% |
| C | 1500 | 90 | 6.0% |

## Key Insight

Campaign C achieved the highest conversion rate and generated the best performance.
